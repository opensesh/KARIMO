# karimo-testing-standards

**Type:** Skill
**Applies to:** karimo-tester, karimo-tester-opus

## Purpose

- Framework detection protocol (Vitest, Jest, Playwright)
- Test structure templates (unit, integration, component)
- Coverage requirements (happy path, errors, edge cases)
- Mocking patterns (module, function, API)
- Assertion reference

## Usage

Tester agents follow these standards when writing tests.
