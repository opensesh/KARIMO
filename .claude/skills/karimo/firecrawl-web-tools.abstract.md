# karimo-firecrawl-web-tools

**Type:** Skill
**Applies to:** karimo-researcher, karimo-refiner

## Purpose

Web scraping, search, and browser automation via Firecrawl MCP server for external research.

## Key Capabilities

- Single page scraping (firecrawl_scrape)
- Web search (firecrawl_search)
- URL discovery (firecrawl_map)
- Multi-URL extraction (firecrawl_extract)
- Browser automation (firecrawl_browser_*)

## Usage

Research agents use Firecrawl for documentation scraping and library evaluation.
