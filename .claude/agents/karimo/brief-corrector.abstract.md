# karimo-brief-corrector

**Type:** Agent
**Model:** sonnet
**Trigger:** After brief-reviewer produces findings with critical issues

## Purpose

Correction agent that applies fixes to task briefs and PRD based on review findings. Modifies briefs, PRD, and tasks.yaml as needed.

## Key Capabilities

- Finding interpretation and prioritization
- Brief modification and correction
- PRD updates for critical issues
- tasks.yaml synchronization
- Change documentation

## Tools

Read, Edit, Write, Grep, Glob
