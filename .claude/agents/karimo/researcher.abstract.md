# karimo-researcher

**Type:** Agent
**Model:** sonnet
**Trigger:** /karimo-research or interviewer spawns for research phase

## Purpose

Conducts two-phase research (internal codebase + external web) to gather implementation context for PRDs.

## Key Capabilities

- Internal codebase pattern discovery
- External documentation scraping (via Firecrawl)
- Library evaluation and comparison
- Best practices research
- Source attribution and findings synthesis

## Tools

Read, Grep, Glob, Bash, WebSearch, WebFetch, Firecrawl MCP
