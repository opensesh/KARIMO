# /karimo-help

**Type:** Command
**Invokes:** None (read-only)

## Purpose

Get help with KARIMO commands or search documentation for specific topics.

## Key Arguments

- (default): List all commands by category
- `{query}`: Search documentation for query

## Categories

- **Core Workflow**: plan, run, dashboard, merge
- **Configuration**: configure (modes and options)
- **Management**: feedback
- **Diagnostics**: doctor (full and test modes)
- **Maintenance**: update, configure --cd

Searches across all `.karimo/docs/*.md` files and command definitions.
