# /karimo-finish — Complete PRD Implementation

Create final PR to main after feature branch execution completes. This finalizes the PRD workflow.

> **Alias for:** `/karimo-merge`
> **Note:** This command uses the same proven merge logic with a clearer name that reflects completing the PRD.

## Usage

```bash
/karimo-finish --prd {slug} [--skip-validation] [--auto-merge]
```

## Arguments

- `--prd {slug}` (required): The PRD slug to finalize.
- `--skip-validation` (optional): Skip validation suite (use with caution).
- `--auto-merge` (optional): Automatically merge PR after creation (requires passing validation).

## What This Command Does

1. **Validates feature branch** — Ensures all tasks merged, no open PRs
2. **Runs validation suite** — Build, typecheck, lint, tests
3. **Analyzes integration** — Detects potential conflicts with main
4. **Creates final PR** — `feature/{prd-slug}` → `main`
5. **Cleans up branches** — After merge completes

## Benefits

- **Single production deployment** — One PR to main for entire PRD
- **Consolidated review** — Review all changes together
- **Clean git history** — Squash all task commits into one
- **No deployment spam** — One deployment instead of 15+

## Example

```bash
# Standard workflow
/karimo-finish --prd user-profiles

# Skip validation (not recommended)
/karimo-finish --prd user-profiles --skip-validation

# Auto-merge after CI passes
/karimo-finish --prd user-profiles --auto-merge
```

## Prerequisites

Before running this command:
1. Complete `/karimo-run --prd {slug}` (orchestration)
2. Wait for all task PRs to merge to feature branch
3. Ensure status is `ready-for-merge`

## Validation Suite

The command runs these checks before creating the final PR:

- ✓ **Build** — Project builds successfully
- ✓ **Typecheck** — No type errors
- ✓ **Lint** — Code style passes
- ✓ **Tests** — All tests passing

If validation fails, fix issues on the feature branch and retry.

## Technical Details

This command delegates to `/karimo-merge` which:
- Checks out feature branch
- Runs validation commands from `.karimo/config.yaml`
- Generates consolidated diff
- Creates final PR with detailed summary
- Optionally auto-merges after CI passes

For full implementation details, see `.claude/commands/karimo-merge.md`.
