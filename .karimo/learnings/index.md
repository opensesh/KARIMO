## How Learnings Work

1. **Capture:** `/karimo-feedback` conducts interview + investigation
2. **Categorize:** Findings are classified by type and severity
3. **Store:** Learning files are created in appropriate category directory
4. **Apply:** Agents read learnings during task execution

## Severity Levels

| Severity | Meaning | Agent Behavior |
|----------|---------|----------------|
| **critical** | Must follow | Hard stop if violated |
| **important** | Should follow | Warning if violated |
| **info** | Nice to know | Optional application |
